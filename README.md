# AI Task Platform

A full-stack AI task processing platform built with the MERN stack, Python worker, Redis queue, Docker, and Kubernetes.

## Architecture

- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Worker**: Python (background job processor)
- **Database**: MongoDB
- **Queue**: Redis

## Local Development

```bash
docker-compose up --build
```
